# Amit's Game Factory V2 🎮

האוטומציה יוצרת **שני משחקים ביום**, שומרת אותם ב-GitHub, מפרסמת בחינם דרך **GitHub Pages**, ושולחת קישור במייל.

## מה השתפר ב-V2

- ארבעה רעיונות מתחרים נוצרים בכל ריצה.
- סוכן מנהל בוחר ומפתח רק את הרעיון החזק ביותר.
- צוות בנייה מייצר משחק מלא עם 8+ שלבים, התנהגויות שונות, שדרוגים ובוס.
- בדיקות טכניות דוחות משחק שבור.
- חבר מושבעים נותן ציון לכיף, עומק, התקדמות, בהירות, עיצוב, מקוריות ומובייל.
- המשחק מקבל מעבר שיפור ממוקד.
- משחק מתחת לציון 78 לא מתפרסם.
- GitHub Pages מחליף את Netlify, ולכן אין קרדיטי העלאה של Netlify.

## לפני ההפעלה

בחשבון GitHub Free, GitHub Pages פועל ממאגר ציבורי. שנה את המאגר ל-Public, ואז פתח:

`Settings → Pages → Build and deployment → Source → GitHub Actions`

לאחר מכן הפעל ידנית:

`Actions → Amit Game Factory V2 → Run workflow`

כתובת האתר תהיה בדרך כלל:

`https://USERNAME.github.io/REPOSITORY/`

## הערה חשובה

AI לא יכול להבטיח שכל משחק יהיה להיט. V2 מחמיר מאוד את תהליך הבחירה, הבנייה והביקורת, ולכן הוא אמור להפיק משחקים טובים משמעותית; משחק שלא עובר את סף האיכות פשוט לא יעלה.
